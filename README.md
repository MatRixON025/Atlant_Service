# Atlant Service — Сайт-візитка

Простий веб-сайт-візитка для сервісного центру з ремонту побутової техніки на Python + Flask.

## 🚀 Швидкий старт

1. Перейдіть у папку проекту:

```powershell
cd "C:\Users\Acer\OneDrive\Desktop\atlant-service"
```

2. Встановіть залежності:

```powershell
python -m pip install -r requirements.txt
```

3. Запустіть додаток:

```powershell
python app.py
```

4. Відкрийте у браузері:

```
http://localhost:5000
```

## 🛠 Налаштування

- **Секретний ключ Flask**: створіть у корені файл `.env` і додайте:

```env
FLASK_SECRET=будь-який_секрет
```

- Змініть тексти на сайті у шаблонах:
  - `templates/index.html`
  - `templates/layout.html`

- Оновіть контактні дані та перелік послуг у `app.py`.

## 🔐 Адмін‑панель

Для перегляду повідомлень, відправлених через форму, введіть у браузері:

```
http://localhost:5000/admin/messages
```

Логін/пароль за замовчуванням (можна змінити у `.env`):

```
ADMIN_USER=admin
ADMIN_PASS=password
```

## 📄 Сторінки сайту

- **Головна** (`/`): загальна інформація, контакти, бренди
- **Послуги** (`/services`): детальний опис послуг з модальними вікнами
- **Адмін** (`/admin/messages`): перегляд заявок (захищено)

## ✅ Що реалізовано

- Сторінка з описом послуг та блоком "Про нас"
- Форма зворотного зв’язку (локально, без реальної відправки email)
- Адаптивна верстка (desktop + mobile)
- Окрема сторінка послуг з модальними вікнами деталей
- Логотипи брендів з модальними вікнами гарантійних випадків
- Адмін-панель для перегляду заявок
